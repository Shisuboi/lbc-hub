Bienvenue sur Leboncoin DealFinder AI

Cet outil utilise Playwright pour extraire les annonces de recherche Leboncoin et fait analyser leur description par votre modèle d'intelligence artificielle local (via Ollama) pour dénicher les meilleures affaires du marché.

1

Recherchez sur LBC

Faites votre recherche sur le site Leboncoin et copiez l'adresse URL du résultat.

2

Configurez l'IA

Déterminez vos critères d'exigence (RAM, état, modèle exact) et choisissez le modèle Ollama local.

3

Laissez faire la magie

L'IA va filtrer les mauvaises annonces et attribuer une note précise de rapport qualité/prix sur 100.

Utiliser une IA en ligne (Claude.ai, Gemini…)

1
Scrapez d'abord les annonces avec le bouton ⚡ Lancer l'Analyse (le modèle Ollama peut être n'importe lequel). Une fois terminé, le fichier leboncoin_brut.json est créé dans le dossier de l'application.
2
Ouvrez Claude.ai, Gemini, ChatGPT ou toute IA en ligne.
Créez un nouveau message et attachez le fichier leboncoin_brut.json.
3
Copiez le prompt ci-dessous (il inclut déjà vos critères) et collez-le dans votre message IA, puis envoyez.
Prompt généré (basé sur vos critères)
4
L'IA va vous retourner un bloc JSON. Copiez-le dans un fichier texte, renommez-le en .json et sauvegardez-le sur votre disque.
5
Revenez ici et cliquez sur 📥 Importer résultats IA (JSON) pour charger le fichier et afficher les résultats.
